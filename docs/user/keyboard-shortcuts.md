# Keyboard Shortcuts

## Global (work from any application)

| Shortcut       | Action                             |
| -------------- | ---------------------------------- |
| `Ctrl+Shift+S` | Open TLDR and paste clipboard text |
| `Ctrl+Shift+X` | Stop reading aloud                 |

## In-App

| Shortcut | Action               |
| -------- | -------------------- |
| `Ctrl+V` | Paste clipboard text |
| `Escape` | Close settings popup |

## Drag and Drop

Drop a `.pdf`, `.docx`, `.txt`, or `.md` file onto the window to load it.
